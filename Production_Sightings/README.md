# Production Sightings

These are anonymized, fictionalized composites, not real leaked company code. But every pattern here has a thousand real siblings running in a thousand real systems right now, and that is worse than if we'd just made them up.

- `sleep_of_shame.py` — a `time.sleep(1)` nobody has dared remove
- `global_flag_of_shame.py` — an admin dashboard's undocumented kill switch
- `validation_theater.py` — "validation" that checks for the presence of an `@`
